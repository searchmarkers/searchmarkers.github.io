Privacy Policy

Terms used in this privacy notice

Extension
The browser extension product, Mark My Search.
Browser
A web browser application on which the Extension is installed.
User
A user of the Extension.

Data policy

The Extension does not collect or store any personal information.

Some non-personal data is stored in the Browser and deleted when the Browser is restarted:

  • Words to be highlighted, gathered spontaneously from search queries and from manual entry by the User.
  • Dynamic bookmarks stored by the User (bookmarks with a replaceable URL field).

Some non-personal data is stored in the Browser, and may be subject to the Browser's sync feature:

  • Explicit extension configuration by the User.
